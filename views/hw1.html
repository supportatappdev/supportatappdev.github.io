  • Create Hellow World App
1.Login into Dennis
2.Click on My Apps Tab, It will show you all the apps created in your Org. Then click on Create App button to create new application
3. Enter app name and description then click on Save button
4. Choose your app from My Apps.
5. To Edit an application select on the Actions Menu then select Edit App option.
6. To create a new file click on New menu item.
7. Add the html content and save the file by Ctrl+s. To preview the application, Click on Actions --> Goto Apps.
8. You will see the changes that you made in the Editor. Thats it!!
