  • Create Custom Objects
1. Custom Objects are data carrying object and bridge betwee UI and database. Each Custom Object bind with one or more data objects in the data base. By using custom object, we can get/update data from/to database in UI. Follow below steps to create custom objects.
2. Open an application in App Editor, Then go to Tools menu and select SQL Editor.
3. Enter DDL to create a table and click on Execute button.
4. Go to Add Details page, click on Custom Objects Tab and click on Create Object Button
5.Select recently created table and enter custom object name click on Generate button.
6. You can see the mapping table and mapping columns for recently created custom object.
